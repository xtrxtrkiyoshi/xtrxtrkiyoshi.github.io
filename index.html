Howl Away
❮ ❯

A mobile application game created using Android Studio and LibGDX. It is a 2D run and jump game about a wolf that escaped a zoo. The game has randomized obstacles, bonuses, coins, and power-ups. It also has a shop for players to choose their wolf. The project was done as a group for an organization requirement. The project proposal was pitched by me due to my interest in gaming, game development, and mobile application development. The project was led by me. I created most of the game assets and handled half of the frontend.
